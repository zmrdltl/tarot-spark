# tarot-spark

Free tarot card drawing and AI prompt generator built with Next.js.

## Development

Use pnpm for local development and CI.

```sh
pnpm install
pnpm dev
```

Run the required quality gates before opening or updating a PR:

```sh
pnpm run format:check
pnpm run lint
pnpm run typecheck
pnpm run test
pnpm run build
```
